SOHO Network Design

Objective: Designed and deployed a secure, high-performance Small Office/Home Office (SOHO) network with optimized performance and robust security protocols.

Network Hardware Configuration:

  • Routers and Switches: Used Ubiquiti routers and Layer 2/3 managed switches from the UniFi product line.
  • VLANs: Segmented network traffic based on device type and function.
  • Wireless Access Points (WAPs): Deployed dual-band UniFi WAPs.

Security Implementations:

  • Next-Generation Firewall (NGFW): pfSense-based firewall with deep packet inspection.
  • Intrusion Detection/Prevention Systems (IDS/IPS): Integrated Snort IDS/IPS.
  • VPN Configuration: Established secure VPN tunnels using OpenVPN with AES-256 encryption.
  • DNS Sinkhole: Configured pfBlockerNG for DNS sinkholing.

Results:

Successfully implemented a secure SOHO network capable of mitigating advanced threats.
