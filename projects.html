SOHO Network Design

Objective: In this project, I meticulously designed and deployed a comprehensive Small Office/Home Office (SOHO) network, with a dual focus on optimizing performance and enforcing robust security protocols.

Network Hardware Configuration:

  • Routers and Switches: Utilized Ubiquiti routers and Layer 2/3 managed switches from the UniFi product line for scalability and high performance.
  • VLANs: Established to segment network traffic based on device type and function.
  • Wireless Access Points (WAPs): Deployed dual-band UniFi WAPs supporting both 2.4 GHz and 5 GHz frequencies.

Security Implementations:

  • Next-Generation Firewall (NGFW): Implemented a pfSense-based NGFW featuring deep packet inspection (DPI), application-layer filtering, and real-time intrusion prevention capabilities.
  • Intrusion Detection/Prevention Systems (IDS/IPS): Integrated Snort IDS/IPS within the pfSense environment.
  • VPN Configuration: Established secure site-to-site and remote access VPN tunnels using OpenVPN, fortified with AES-256 encryption.
  • DNS Sinkhole: Configured pfBlockerNG on pfSense to act as a DNS sinkhole.

Results:

Successfully implemented a secure SOHO network capable of mitigating advanced threats while optimizing performance.
