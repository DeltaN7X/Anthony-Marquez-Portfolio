SOHO Network Design

Objective: In this project, I meticulously designed and deployed a comprehensive Small Office/Home Office (SOHO) network, with a dual focus on optimizing performance and enforcing robust security protocols. The project commenced with a thorough assessment of network requirements, involving detailed inventorying of all devices, precise bandwidth allocation analysis, and an in-depth evaluation of necessary security measures to safeguard data integrity and confidentiality.

Network Hardware Configuration:

  • Routers and Switches: Utilized Ubiquiti routers and Layer 2/3 managed switches from the UniFi product line for scalability and high performance. Configured dynamic routing protocols like OSPF using pfSense to optimize path selection and ensure efficient data flow.
  • VLANs: Established to segment network traffic based on device type and function, enhancing both performance and security by isolating sensitive data streams.
  • Wireless Access Points (WAPs): Deployed dual-band UniFi WAPs supporting both 2.4 GHz and 5 GHz frequencies, configured with WPA3 encryption for enhanced wireless security. SSID segmentation was employed to create separate networks for guests, IoT devices, and internal systems, minimizing potential attack vectors.

Security Implementations:

  • Next-Generation Firewall (NGFW): Implemented a pfSense-based NGFW featuring deep packet inspection (DPI), application-layer filtering, and real-time intrusion prevention capabilities. Defined granular Access Control Lists (ACLs) to manage network permissions and restrict unauthorized access.
  • Intrusion Detection/Prevention Systems (IDS/IPS): Integrated Snort IDS/IPS within the pfSense environment with customized rule sets to detect and mitigate a variety of network threats, including port scanning, SQL injection attempts, and brute-force login attacks.
  • VPN Configuration: Established secure site-to-site and remote access VPN tunnels using OpenVPN, fortified with AES-256 encryption to ensure data confidentiality.
  • DNS Sinkhole: Configured pfBlockerNG on pfSense to act as a DNS sinkhole, intercepting and redirecting traffic from known malicious domains to a controlled environment, effectively mitigating phishing attempts and malware infections.

Monitoring and Maintenance:

  • Network Monitoring: Conducted using Wireshark for deep packet-level analysis and Zenmap for network scanning and vulnerability detection. These tools enabled detailed traffic inspection and network mapping to identify potential security risks and optimize network performance.
  • Regular Audits: Implemented a structured schedule for regular firmware updates and security patches to maintain the integrity, stability, and security of the network infrastructure.

Results:

Successfully implemented a secure SOHO network capable of mitigating advanced threats while optimizing performance. Regular security audits and firmware updates ensured the integrity and stability of the infrastructure.

Active Directory Administration

Objective: This project entailed the comprehensive administration and optimization of Active Directory (AD) services to streamline user management workflows, enhance access control mechanisms, and reinforce organizational security frameworks.

User Account Management:

  • Automation with PowerShell: Developed extensive PowerShell scripts to automate the entire user lifecycle management process, including account creation, group membership assignments, password policy enforcement, and deprovisioning.

Group Policy Management:

  • GPO Configuration: Configured and enforced Group Policy Objects (GPOs) to standardize security settings across the domain, including password complexity requirements, account lockout thresholds, software installation restrictions, and desktop environment configurations.

Access Control and Permissions:

  • NTFS Permissions Management: Administered NTFS file system permissions to control access to critical data, rigorously applying the principle of least privilege. Implemented role-based access controls (RBAC) to simplify permission management and ensure scalability.

Auditing and Monitoring:

  • Audit Trails: Enabled advanced auditing features in AD to monitor critical changes to user accounts, group memberships, and security settings. Maintained comprehensive audit trails to support forensic investigations, compliance audits, and security incident responses.
  • Secure Remote Access: Configured Remote Access Services (RAS) with Network Address Translation (NAT) and Port Address Translation (PAT) to manage IP address allocation and ensure secure data transmission.
  • Multi-Factor Authentication (MFA): Integrated MFA mechanisms combining traditional credentials with biometric verification and one-time passcodes (OTPs), significantly mitigating the risk of unauthorized access.

Maintenance and Troubleshooting:

  • Conducted regular reviews of Active Directory event logs to promptly identify and rectify potential issues. Performed periodic security audits to ensure adherence to best practices and compliance with organizational policies.
  • Provided technical support for complex AD-related issues, ensuring minimal disruption to business operations.
  • Implemented proactive monitoring solutions and conducted capacity planning to anticipate future growth and resource requirements.

Results:

Improved operational efficiency by automating user management tasks, enhancing file server security, and supporting secure remote access for staff, resulting in minimized disruptions and strengthened overall security posture.
